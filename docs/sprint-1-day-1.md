# Sprint 1 - Day 1

## Goal

Create the foundation for a running HR application with login, CV upload, candidate storage, search, and duplicate detection. AI is intentionally excluded from this sprint.

## CV Processing Flow

```text
Upload
-> Extract Text
-> Extract Candidate
-> Check Duplicate
-> Create Candidate
-> Store CV
-> Done
```

## Duplicate Engine Priority

1. Phone
2. Email
3. National ID
4. Name Similarity
5. Company History

National ID is reserved for a future candidate field.
